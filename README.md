# 🛒 E-Commerce Sales Analysis using SQL

## 📌 Overview

This project analyzes e-commerce sales data using SQL to generate business insights related to customers, products, revenue, and sales trends.

The project demonstrates relational database design, SQL joins, aggregate functions, grouping, sorting, and business reporting.

---

## 🎯 Objectives

- Analyze customer purchasing behavior
- Identify top-selling products
- Calculate total revenue
- Compare category-wise sales
- Find high-value customers
- Generate business insights using SQL

---

## 🛠 Tech Stack

- MySQL
- SQL
- Excel
- Relational Database

---

## 📂 Dataset

The project contains three related tables:

- Customers
- Products
- Orders

---

## 📊 SQL Concepts Used

- Primary Keys
- Foreign Keys
- INNER JOIN
- GROUP BY
- HAVING
- ORDER BY
- Aggregate Functions
- Business Queries
